1.1.7. 系统优化 ​

bash
# 优化系统性能
echo "vm.swappiness = 1" >> /etc/sysctl.conf
sysctl -p

# 优化磁盘性能
echo "vm.vfs_cache_pressure = 50" >> /etc/sysctl.conf
sysctl -p

# 优化网络性能
echo "net.core.rmem_max = 65536" >> /etc/sysctl.conf
echo "net.core.wmem_max = 65536" >> /etc/sysctl.conf
echo "net.core.rmem_default = 65536" >> /etc/sysctl.conf
echo "net.core.wmem_default = 65536" >> /etc/sysctl.conf
echo "net.core.optmem_max = 40960" >> /etc/sysctl.conf
echo "net.ipv4.tcp_rmem = 4096 87380 65536" >> /etc/sysctl.conf
echo "net.ipv4.tcp_wmem = 4096 65536 87380" >> /etc/sysctl.conf
echo "net.ipv4.tcp_low_latency = 1" >> /etc/sysctl.conf
echo "net.ipv4.tcp_adv_win_scale = 1" >> /etc/sysctl.conf
echo "net.ipv4.tcp_congestion_control = cubic" >> /etc/sysctl.conf
sysctl -p

1.2. 常用工具 ​

1.2.1. 系统监控 ​

  • top：实时显示系统资源使用情况
  • htop：增强版的top，支持颜色、排序等功能
  • vmstat：显示系统虚拟内存、进程、中断、CPU 等状态
  • iostat：显示 CPU 使用率和 I/O 状态
  • netstat：显示网络连接、路由表、接口统计等
  • iftop：实时显示网络带宽使用情况
  • ss：显示套接字统计信息
  • lsof：列出系统打开的文件

1.2.2. 日志分析 ​

  • grep：搜索日志文件
  • awk：分析日志文件
  • sed：流编辑器，对日志文件进行替换、删除、替换等操作
  • tail：显示文件末尾内容
  • head：显示文件开头内容
  • logrotate：日志轮转，对日志文件进行切割、压缩等操作

1.2.3. 性能分析 ​

  • strace：跟踪系统调用
  • tcpdump：抓取网络数据包
  • perf：分析系统性能瓶颈
  • dstat：实时显示系统资源使用情况
  • sysbench：测试数据库性能

1.2.4. 系统管理 ​

  • systemctl：管理系统服务

2. 常见问题 ​

2.1. 系统问题 ​

2.1.1. 系统卡顿 ​

  • 检查系统资源使用情况，如 CPU、内存、磁盘、网络等
  • 检查系统日志，查看是否有错误信息
  • 检查系统服务，查看是否有异常服务
  • 检查系统配置，查看是否有优化空间

2.1.2. 系统崩溃 ​

  • 检查系统日志，查看是否有错误信息
  • 检查系统服务，查看是否有异常服务
  • 检查系统配置，查看是否有优化空间
  • 重启系统，排除故障
