deploy-vitepres-to-gitee ​

前言 ​

VitePress 是一个基于 Vue 和 Markdown 的静态站点生成器。本文将介绍如何将 VitePress 部署到 Gitee Pages。

准备工作 ​

首先，你需要拥有一个 Gitee 账号，并安装 Git。如果你还没有，请访问 Gitee 注册页面 进行注册。

然后，你需要安装 VitePress 脚手架。如果你还没有，请运行以下命令安装：

bash
npm install -g vitepress
1

创建项目 ​

创建一个名为 my-vitepress-blog 的文件夹，并进入该文件夹：

bash
mkdir my-vitepress-blog && cd my-vitepress-blog
1

然后，运行以下命令创建 VitePress 项目：

bash
vitepress init
1

配置项目 ​

打开 vitepress.config.js 文件，修改 base 配置项，将其设置为你的 Gitee 用户名。

js
module.exports = {
  base: '/my-vitepress-blog/', // 修改此处
  lang: 'zh-CN',
  title: 'My VitePress Blog',
  description: 'Just playing around',
  head: [['link', {rel: 'icon', href: '/logo.svg'}]],
  themeConfig: {
    repo: 'https://gitee.com/your-username/my-vitepress-blog', // 修改此处
    editLinks: true,
    editLinkText: '在 Gitee 上编辑此页',
    lastUpdated: '上次更新',
    nav: [
      {text: '首页', link: '/'},
      {text: '关于', link: '/about/'},
      {text: '标签', link: '/tags/'},
    ],
    sidebar: {
      '/': [
        {
          text: '介绍',
          link: '/',
          children: [
            {text: '介绍', link: '/'},
            {text: '开始', link: '/start/'},
            {text: '目录结构', link: '/structure/'},
          ],
        },
        {
          text: '指南',
          link: '/guide/',
          children: [
            {text: 'Markdown 语法', link: '/guide/markdown/'},
            {text: '目录结构', link: '/guide/structure/'},
            {text: '配置项', link: '/guide/config/'},
          ],
        },
      ],
      '/about/': [
        {
          text: '关于',
          link: '/about/',
          children: [
            {text: '介绍', link: '/about/'},
            {text: '作者', link: '/about/author/'},
            {text: '鸣谢', link: '/about/thanks/'},
          ],
        },
      ],
      '/tags/': [
        {
          text: '标签',
          link: '/tags/',
          children: [{text: '标签', link: '/tags/'}],
        },
      ],
    },
  },
};

编写内容 ​

在 docs 文件夹下，编写你的文章。

部署项目 ​

在项目根目录下，运行以下命令将项目部署到 Gitee Pages：

bash
npm run docs:build
1

然后，将 docs/.vitepress/dist 文件夹下的内容上传到 Gitee Pages。

访问项目 ​

打开浏览器，访问 https://your-username.gitee.io/my-vitepress-blog/，就可以看到你的 VitePress 博客了。

后记 ​

本文介绍了如何将 VitePress 部署到 Gitee Pages。Gitee Pages 是基于 Git 的静态站点托管服务，它提供免费的域名和 SSL 证书，并且支持自定义域名。如果你想使用自定义域名，请访问 Gitee Pages 自定义域名文档 进行配置。

VitePress 是一个很棒的静态站点生成器，它提供了简洁、高效的开发体验，并且支持 Markdown 语法。如果你想了解更多关于 VitePress 的信息，请访问 VitePress 官方文档。
