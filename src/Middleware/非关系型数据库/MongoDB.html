MongoDB ​

MongoDB 是一种基于分布式文件存储的数据库。它是一个开源的数据库软件，由 C++ 语言编写。

安装 MongoDB ​

MongoDB 可以在 Linux、Windows 和 macOS 上安装。

Linux ​

在 Linux 上安装 MongoDB 可以使用包管理器。例如，在 Ubuntu 上可以使用以下命令安装：

sudo apt-get install mongodb
1

Windows ​

在 Windows 上安装 MongoDB 可以从 MongoDB 官网下载安装程序。下载完成后，双击运行安装程序，按照提示进行安装。

macOS ​

在 macOS 上安装 MongoDB 可以使用 Homebrew。首先，安装 Homebrew：

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"
1

然后，使用 Homebrew 安装 MongoDB：

brew install mongodb
1

配置 MongoDB ​

MongoDB 默认安装完成后，会自动创建数据目录 /data/db，用于存储数据文件。

如果需要修改数据目录，可以修改配置文件 mongod.conf。

dbpath = /path/to/data/directory
1

启动 MongoDB ​

启动 MongoDB 可以使用命令 mongod。

mongod
1

如果需要在后台运行 MongoDB，可以使用命令 mongod --fork。

mongod --fork
1

连接 MongoDB ​

连接 MongoDB 可以使用命令 mongo。

mongo
1

基本操作 ​

创建数据库 ​

使用命令 use 可以创建数据库。

use mydb
1

创建集合 ​

使用命令 db.createCollection("mycollection") 可以创建集合。

db.createCollection("mycollection")
1

插入数据 ​

使用命令 db.mycollection.insert({name: "John", age: 30}) 可以插入数据。

db.mycollection.insert({name: "John", age: 30})
1

查询数据 ​

使用命令 db.mycollection.find() 可以查询数据。

db.mycollection.find()
1

更新数据 ​

使用命令 db.mycollection.update({name: "John"}, {name: "Jane", age: 35}) 可以更新数据。

db.mycollection.update({name: "John"}, {name: "Jane", age: 35})
1

删除数据 ​

使用命令 db.mycollection.remove({name: "Jane"}) 可以删除数据。

db.mycollection.remove({name: "Jane"})
1
