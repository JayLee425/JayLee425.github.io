Vue Base ​

1. Vue 基础 ​

  • Vue 实例的创建

    • 创建一个 Vue 实例: new Vue({...options})。
    • 数据绑定: data 属性。
    • 生命周期钩子: created、mounted、updated、destroyed。
javascript
new Vue({
  el: '#app',
  data: {
    message: 'Hello Vue!',
  },
  created: function () {
    console.log('created');
  },
  mounted: function () {
    console.log('mounted');
  },
  updated: function () {
    console.log('updated');
  },
  destroyed: function () {
    console.log('destroyed');
  },
});

  • el 属性: 指定 Vue 实例的挂载点，可以是 CSS 选择器、HTML 元素或一个 DOM 元素。
  • data 属性: 用于存储 Vue 实例的数据。

  • 生命周期钩子: created、mounted、updated、destroyed。

    • created: 在实例创建之后调用，实例已经完成了初始化，但还没有开始渲染过程。
    • mounted: 在实例挂载之后调用，实例已完成渲染过程，可以进行 DOM 操作。
    • updated: 在数据更新时调用，发生在虚拟 DOM 重新渲染和打补丁之前。
    • destroyed: 在实例销毁之前调用，实例完全销毁，可以进行一些清理工作。

2. 模版语法 ​

  • 插值: 。
  • 指令:
    • v-bind: 绑定属性，简写 :。
    • v-model: 双向绑定，绑定表单元素和数据。
    • v-if、v-else, v-show: 条件渲染。
    • v-for: 列表渲染。
    • v-on: 绑定事件, 简写 @。
html
<div id="app">
  <p>{{ message }}</p>
  <input type="text" v-model="message" />
  <ul>
    <li v-for="item in items" v-bind:key="item.id">{{ item.text }}</li>
  </ul>
  <button v-on:click="reverseMessage">Reverse Message</button>
</div>

3. 计算属性和侦听器 ​

  • 计算属性: computed, 基于依赖缓存。
  • 侦听器: watch 属性, 监听数据变化。
javascript
new Vue({
  el: '#app',
  data: {
    message: 'Hello Vue!',
    items: [
      {id: 1, text: 'Item 1'},
      {id: 2, text: 'Item 2'},
      {id: 3, text: 'Item 3'},
    ],
  },
  computed: {
    reversedMessage: function () {
      return this.message.split('').reverse().join('');
    },
  },
  watch: {
    message: function (newVal, oldVal) {
      console.log('Message changed from ' + oldVal + 'to ' + newVal);
    },
  },
  methods: {
    reverseMessage: function () {
      this.message = this.reversedMessage;
    },
  },
});

5. Vue Router (路由) ​

  • 路由配置：定义路由和组件映射。

  • 路由导航：<router-link>, this.$router.push。

  • 动态路由：/user/:id。

  • 嵌套路由：子路由。

javascript
// 路由配置
const routes = [
  {path: '/', component: Home},
  {path: '/user/:id', component: User},
  {path: '/about', component: About},
];

// 创建路由实例
const router = new VueRouter({
  routes,
});

// 创建 Vue 实例并使用路由
new Vue({
  router,
  render: (h) => h(App),
}).$mount('#app');

6. Vuex (状态管理) ​

  • 状态管理: 集中式存储管理。

  • 状态: 全局共享的状态。

  • 状态树: 多个组件共享的状态。

  • 单一状态树: 集中式存储管理。

  • mutation: 同步修改状态树。

  • action: 异步修改状态树。

  • getter: 获取状态树。

  • module: 按模块划分状态树。

javascript
// 创建 Vuex 实例
const store = new Vuex.Store({
  state: {
    count: 0,
  },
  mutations: {
    increment(state) {
      state.count++;
    },
  },
  actions: {
    incrementAsync(context) {
      setTimeout(() => {
        context.commit('increment');
      }, 1000);
    },
  },
  getters: {
    doubleCount(state) {
      return state.count * 2;
    },
  },
});

// 创建 Vue 实例并使用 Vuex
new Vue({
  el: '#app',
  store,
  data: {
    message: 'Hello Vue!',
  },
  computed: {
    reversedMessage: function () {
      return this.message.split('').reverse().join('');
    },
  },
  watch: {
    message: function (newVal, oldVal) {
      console.log('Message changed from ' + oldVal + 'to ' + newVal);
    },
  },
  methods: {
    reverseMessage: function () {
      this.message = this.reversedMessage;
    },
    increment: function () {
      this.$store.commit('increment');
    },
    incrementAsync: function () {
      this.$store.dispatch('incrementAsync');
    },
  },
});

7. Vue CLI (脚手架) ​

  • 创建项目: vue create my-project。

  • 运行项目: npm run serve。

  • 构建项目: npm run build。
