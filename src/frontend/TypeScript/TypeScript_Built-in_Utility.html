> 1. Awaited ​

Awaited<T> 类型用来获取 T 类型中所有 Promise 类型的值。

ts
type Awaited<T> = T extends Promise<infer U> ? U : T;
1

例如：

ts
type T = Promise<string> | string;
type R = Awaited<T>; // string

> 2. Partial ​

Partial<T> 类型用来将 T 类型中的所有属性设为可选。

ts
type Partial<T> = {
  [P in keyof T]?: T[P];
};
1
2
3

例如：

ts
interface Person {
  name: string;
  age: number;
}

type PartialPerson = Partial<Person>;

>5. Record ​

Record<K, T> 类型用来创建一个对象，其属性名为 K 类型的值，属性值为 T 类型的值。

ts

>18. Conditional type ​

T extends U? X : Y 类型用来根据 T 类型是否是 U 类型来决定 X 类型还是 Y 类型。

ts
type Conditional<T extends U, U, X, Y> = T extends U ? X : Y;
1

例如：

ts
type T = 1 | 2 | 3;
type R = Conditional<T, 2, 'yes', 'no'>; // 'yes'

>19. Type inference ​

TypeScript 能够根据类型推断来自动推导出类型，这使得代码更加简洁。

ts
let x = 1;
let y = 'hello';

x = y; // error: Type 'string' is not assignable to type 'number'.
1
2
3
4

上面的代码中，TypeScript 能够推导出 x 的类型为 number，因为 y 的类型为 string，而 number 不能赋值给 string。

ts
let x = 1;
let y = 'hello';

x = y as number; // ok
1
2
3
4

上面的代码中，通过 as 关键字，TypeScript 能够将 y 的类型转换为 number，从而解决类型不匹配的问题。

ts
let x: number | string = 1;
let y: string = 'hello';

x = y; // ok
1
2
3
4

上面的代码中，TypeScript 能够推导出 x 的类型为 number | string，因为 y 的类型为 string，而 number 可以赋值给 string。

>22. Type inference in conditional types ​

TypeScript 能够在条件类型中进行类型推断，这使得代码更加简洁。

ts
type First<T extends any[]> = T extends [infer F, ...any[]] ? F : never;
1

上面的代码中，First 条件类型能够根据 T 类型是否是数组来决定 F 类型。

ts
type First<T extends any[]> = T extends [infer F, ...any[]] ? F : never;

type T1 = First<[1, 2, 3]>; // 1
type T2 = First<[]>; // never

类型 ​

>1. 基本类型 ​

  • number：数字类型，包括整数和浮点数。
  • string：字符串类型，包括单引号和双引号。
  • boolean：布尔类型，包括 true 和 false。
  • null：空类型，表示没有值。
  • undefined：未定义类型，表示没有赋值。
  • symbol：符号类型，表示唯一的值。
  • bigint：大整数类型，表示大于 Number.MAX_SAFE_INTEGER 的整数。

>2. 数组类型 ​

>20. 类型保护与类型别名 ​

类型保护与类型别名可以结合使用，用来定义更复杂的类型。

ts
type Name = string;
type Age = number;

function isString(x: any): x is string {
  return typeof x === 'string';
}

type NameOrAge = Name | Age;

function getNameOrAge(x: NameOrAge): NameOrAge {
  if (isString(x)) {
    return x;
  } else {
    return x;
  }
}

let result1: NameOrAge = 'Alice';
let result2: NameOrAge = 25;
