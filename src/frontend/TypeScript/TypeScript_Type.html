TypeScript 类型 ​

基础类型 ​

TypeScript 支持 JavaScript 的所有基础类型，并在此基础上增加了额外的类型检查功能。以下是 TypeScript 中的基础类型：

  1. Boolean: 布尔值，true 或 false

    ts
    let isDone: boolean = false;
    1

  2. Number: 数字类型，包括整数和浮点数

    ts
    let decimal: number = 6;
let hex: number = 0xf00d;
let binary: number = 0b1010;
let octal: number = 0o744;
    1
    2
    3
    4

  3. String: 字符串类型

    ts
    let color: string = 'blue';
color = 'red';
let fullName: string = `Bob`;
let age: number = 37;
let sentence: string = `Hello, my name is ${fullName}. I'll be ${age + 1} years old next month.`;
    1
    2
    3
    4
    5

  4. Array: 数组类型，有两种表示方式

    ts
    let list: number[] = [1, 2, 3];
let list2: Array<number> = [1, 2, 3];
    1
    2

  5. Tuple: 元组类型，允许表示一个已知元素数量和类型的数组

    ts
    let x: [string, number];
x = ['hello', 10]; // OK
x = [10, 'hello']; // Error
    1
    2
    3

  6. Enum: 枚举类型

    ts
    enum Color {
  Red,
  Green,
  Blue,
}
let c: Color = Color.Green;
    1
    2
    3
    4
    5
    6

  7. Any: 任意类型，用于不确定类型的变量

    ts
    let notSure: any = 4;
notSure = 'maybe a string instead';
notSure = false; // okay, definitely a boolean
    1
    2
    3

  8. Void: 无类型，通常用于函数返回值

    ts
    function warnUser(): void {
  console.log('This is my warning message');
}
    1
    2
    3

  9. Null 和 Undefined:

    ts
    let u: undefined = undefined;
let n: null = null;
    1
    2

  10. Never: 表示永不存在的值的类型

    ts
    function error(message: string): never {
  throw new Error(message);
}
    1
    2
    3

  11. Object: 非原始类型

    ts
    declare function create(o: object | null): void;
create({prop: 0}); // OK
create(null); // OK
create(42); // Error

  12. Unknown: 未知类型，比 any 更安全

    ts

value = true; // OK
value = 42; // OK
value = 'Hello World'; // OK
value = []; // OK
value = {}; // OK
value = Math.random; // OK
value = null; // OK
value = undefined; // OK
value = new TypeError(); // OK
value = Symbol('type'); // OK

    ts
    let value: any;
    1

  14. void: 表示没有类型

    ts
    function warnUser(): void {
  console.log('This is my warning message');
}
    1

  15. never: 表示永不存在的值的类型

    ts
    function error(message: string): never {
  throw new Error(message);

let sym2 = Symbol('key2');

    ts
    let n: null = null;

  19. undefined: 表示未定义的值

    let u: undefined = undefined;

这些基础类型为 TypeScript 提供了强大的类型检查能力，帮助开发者在编码阶段发现潜在的错误。

复合类型 ​

复合类型是由基础类型组合而成的更复杂类型，主要包括以下几种：

  1. 数组类型：表示相同类型元素的集合

    ts
    let arr1: number[] = [1, 2, 3];
let arr2: Array<number> = [1, 2, 3]; // 泛型语法
    1
    2

  2. 对象类型：表示具有特定属性和类型的对象

    ts
    let obj: {name: string; age: number} = {
  name: 'Alice',
  age: 25,
};
    1
    2
    3
    4

  3. 函数类型：表示函数的参数和返回值类型

    ts
    let add: (x: number, y: number) => number = (x, y) => x + y;
    1

  4. 元组类型：表示固定长度和类型的数组

    ts
    let tuple: [string, number] = ['Alice', 25];
    1

  5. 联合类型：表示可以是多种类型中的一种

    ts
    let value: string | number = 'Hello';
value = 42; // OK
    1
    2

  6. 交叉类型：表示同时具有多种类型的特性

    ts
    interface A {
  a: string;
}
interface B {
  b: number;
}
type C = A & B;
let obj: C = {a: 'hello', b: 42};
    1
    2
    3
    4
    5
    6
    7
    8

  7. 字面量类型：表示具体的值作为类型

    ts
    let direction: 'left' | 'right' | 'up' | 'down' = 'left';
    1

  8. 映射类型：基于现有类型创建新类型

    ts
    type Readonly<T> = {
  readonly [P in keyof T]: T[P];
};
    1
    2
    3

  9. 条件类型：根据条件选择类型

    ts
    type IsString<T> = T extends string ? true : false;
    1

  10. 索引类型：通过索引访问类型

    ts
    type Person = {name: string; age: number};
type NameType = Person['name']; // string
    1
    2

这些复合类型为 TypeScript 提供了更强大的类型表达能力，使得类型系统更加灵活和精确。

函数类型 ​

TypeScript 中的函数类型用于定义函数的参数类型和返回值类型。以下是函数类型的几种常见用法：

  1. 函数声明：

    ts
    function add(x: number, y: number): number {
  return x + y;
}
    1
    2
    3

  2. 箭头函数：

    ts
    const add = (x: number, y: number): number => x + y;
    1

  3. 函数类型接口：

    ts
    interface MathOperation {
  (x: number, y: number): number;
}
    1
    2
    3

  4. 函数重载：

    ts
    function add(x: number, y: number): number;
function add(x: string, y: string): string;
    1
    2

  5. 可选参数：

    ts
    function greet(name: string, age?: number): string {
  return `Hello, ${name}!`;
}
    1
    2
    3

  6. 默认参数：

    ts
    function greet(name: string, age: number = 20): string {
  return `Hello, ${name}!`;
}
    1
    2
    3

  7. 剩余参数：

    ts
    function sum(...args: number[]): number {
  return args.reduce((acc, curr) => acc + curr, 0);
}
    1
    2
    3

    剩余参数解析： 剩余参数允许我们将不定数量的参数表示为一个数组。这在处理不确定数量参数的函数时非常有用。

    ts
    // 使用剩余参数计算所有数字的和
function sum(...numbers: number[]): number {
  return numbers.reduce((acc, curr) => acc + curr, 0);
}

console.log(sum(1, 2, 3)); // 输出: 6
console.log(sum(10, 20, 30, 40)); // 输出: 100
    1
    2
    3
    4
    5
    6
    7

    剩余参数的特点：

    • 使用 ... 语法表示
    • 必须放在参数列表的最后
    • 类型必须是一个数组类型
    • 可以与其他参数一起使用，但剩余参数必须是最后一个

    示例：

    ts
    function createUser(name: string, ...roles: string[]) {
  console.log(`User ${name} has roles: ${roles.join(', ')}`);
}

createUser('Alice', 'admin', 'editor');
// 输出: User Alice has roles: admin, editor
    1
    2
    3
    4
    5
    6

    剩余参数的类型推断：

    ts
    function sum(...numbers: number[]): number {
  return numbers.reduce((acc, curr) => acc + curr, 0);
}
    1
    2
    3

这些函数类型为 TypeScript 提供了灵活的函数定义方式，使得函数参数和返回值类型更加明确和安全。

交叉类型 ​

交叉类型（Intersection Types）是将多个类型合并为一个类型，使用 & 操作符表示。它允许我们将多个类型的所有特性组合在一起，形成一个新的类型。

ts
type A = {a: string};
type B = {b: number};
type C = A & B;

const obj: C = {a: 'hello', b: 42};

联合类型 ​

联合类型（Union Types）允许一个值可以是多种类型中的一种。使用 | 符号将多个类型组合在一起。

ts
type UnionType = string | number;

const value: UnionType = 'Hello';

类型别名 ​

类型别名（Type Aliases）允许我们为类型创建一个新名称，方便在代码中重复使用。

类型推断 ​

类型推断（Type Inference）允许我们在不显式指定类型的情况下自动推断出变量的类型。

ts
let x = 10; // 推断为 number
let y = 'Hello'; // 推断为 string
let z = [1, 2, 3]; // 推断为 number[]

类型保护 ​

类型保护（Type Guards）允许我们在运行时检查变量的类型，以便在不同的类型之间进行安全的转换。

ts
function isString(value: unknown): value is string {
  return typeof value === 'string';
}

if (isString(value)) {
  console.log(value.toUpperCase());
}
