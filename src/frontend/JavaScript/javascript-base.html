JavaScript Base(基础) ​

1. 变量 ​

1.1 变量声明 ​

javascript
var x = 10;
let y = 20;
const z = 30;
1
2
3
  • var 声明的变量可以重复声明，而 let 和 const 声明的变量不能重复声明。
  • let 和 const 声明的变量只能在块级作用域内使用，不能在函数内使用。
  • const 声明的变量的值不能被修改。

1.2 变量类型 ​

JavaScript 有 6 种基本数据类型：

  • 数值（Number）
  • 字符串（String）
  • 布尔值（Boolean）
  • 数组（Array）
  • 对象（Object）
  • 正则表达式（RegExp）

10.17 事件循环 ​

JavaScript 引擎在执行代码时，会创建事件循环，它会不断检查是否有事件需要处理，如果有，则会将事件添加到事件队列中，等待 JavaScript 引擎的处理。

  • 宏任务（macrotask）：可以理解为可以中断的任务，包括定时器、异步 I/O、UI 渲染等。
  • 微任务（microtask）：可以理解为不可以中断的任务，包括 process.nextTick、Promise、Object.observe、MutationObserver 等。
