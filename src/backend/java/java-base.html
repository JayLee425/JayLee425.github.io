3.Java 控制语句 ​

Java 语言支持以下控制语句：

  • if-else 语句
  • switch-case 语句
  • for 循环语句
  • while 循环语句
  • do-while 循环语句
  • break 语句
  • continue 语句
  • return 语句

注意：Java 语言不支持 goto 语句。

break 和 continue 和 return语句的使用范围：

  • break：循环体中使用，用于 (跳出的循环)。

  • continue：循环体中使用，用于，继续执行下一次循环()逻辑。

  • return：方法体中使用，用于，并的执行。

注意：return语句必须在方法体中使用，不能在类体中使用。

6.Java 异常处理 ​

Java 语言支持异常处理机制，可以用来处理程序运行过程中出现的错误。

Java 异常处理分为两种：

  • 受检异常（checked exception）：必须对其进行处理，否则程序无法正常运行。
  • 非受检异常（unchecked exception）：不需要对其进行处理，可以继续运行。
  1. 异常的声明
java

    // 可能产生异常的代码

  1. 线程的同步
java
public class MyThread extends Thread{    // 继承Thread类
    private int count = 10;    // 共享变量

    public void run(){    // 重写run方法
        for(int i=0; i<count; i++){
            System.out.println(getName() + " " + i);

        }

MyThread thread1 = new MyThread();    // 创建线程1
MyThread thread2 = new MyThread();    // 创建线程2

thread1.start();    // 启动线程1
thread2.start();    // 启动线程2

try{
    thread1.join();    // 等待线程1执行完毕
    thread2.join();    // 等待线程2执行完毕
}catch(InterruptedException e){
    e.printStackTrace();
}

12.Java 序列化 ​

Java 语言支持序列化，可以将对象转换为字节序列，并在另一个程序中恢复。

  1. 序列化的基本概念
  • 序列化：Serialization，序列化，将对象转换为字节序列。
  • 反序列化：Deserialization，反序列化，将字节序列恢复为对象。
  1. 序列化的实现
java
import java.io.Serializable;

public class MyClass implements Serializable {
    private static final long serialVersionUID = 1L;    // 序列化版本号

    private String field;

    public MyClass(String field) {
        this.field = field;
    }

    public String getField() {
        return field;
    }

    public void setField(String field) {
        this.field = field;
    }

    public static void main(String[] args) {
        try {
            // 创建对象
            MyClass object = new MyClass("Hello, world!");

            // 序列化对象
            byte[] bytes = serialize(object);

            // 反序列化对象
            MyClass newObject = (MyClass) deserialize(bytes);

            // 输出字段的值
            System.out.println(newObject.getField());
        } catch (Exception e) {
            e.printStackTrace();
        }
    }

    // 序列化对象
    private static byte[] serialize(Object object) throws Exception {
        try (ObjectOutputStream outputStream = new ObjectOutputStream(new FileOutputStream("object.ser"))) {
            outputStream.writeObject(object);
            outputStream.flush();
            return outputStream.toByteArray();
        }
    }

    // 反序列化对象
    private static Object deserialize(byte[] bytes) throws Exception {
        try (ObjectInputStream inputStream = new ObjectInputStream(new FileInputStream("object.ser"))) {
            return inputStream.readObject();
        }
    }
}
